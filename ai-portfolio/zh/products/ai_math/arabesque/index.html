项目概览

Arabesque 是一个交互式公式可视化平台。用户输入自然语言或 LaTeX 数学公式，AI 自动解析并生成结构化的可视化配置，前端通过 Plotly.js 实时渲染 3D 曲面、参数曲线、向量场、粒子动画等图形，同时提供中文讲解。

技术栈：React + TypeScript + Vite（前端）| Python FastAPI（后端）| Anthropic SDK + OpenAI SDK（AI）| Plotly.js（可视化）

功能模块

1. 公式输入与 AI 解析

输入方式：自然语言（如 z = sin(x) * cos(y)）或描述性文本（如 projectile motion）

AI 双通道（前端可切换）：

通道协议说明
anthropicAnthropic Messages API兼容所有 Anthropic 协议服务（如 MiMo），用 tool_use 强制结构化输出
openaiOpenAI Chat Completions兼容所有 OpenAI 协议服务，用 function calling 强制结构化输出

AI 输出结构：

{
  "type": "surface",
  "expression": "Math.sin(x) * Math.cos(y)",
  "variables": ["x", "y"],
  "domain": { "x": [-5, 5], "y": [-5, 5] },
  "parameters": [{ "name": "A", "label": "振幅", "min": 0.1, "max": 5, "step": 0.1, "default": 1 }],
  "animation": { "variable": "t", "speed": 1.0, "range": [0, 10], "loop": true },
  "explanation": "这是一个二元正弦余弦乘积曲面...",
  "latex": "z = \\sin(x) \\cdot \\cos(y)",
  "title": "sin(x)cos(y) 曲面",
  "dimension": "3d"
}

可视化类型：surface（曲面）| curve（参数曲线）| parametric2d（2D参数图）| vectorField（向量场）| particles（粒子）

2. 3D/2D 可视化渲染

使用 Plotly.js（react-plotly.js）渲染，无需 WebGL，在所有浏览器环境均可工作。

类型Plotly trace说明
surfacetype: 'surface' 或 'contour'3D 曲面 / 2D 等高线图，Viridis 色彩映射
curvetype: 'scatter3d' 或 'scatter'参数曲线，支持 2D/3D
vectorFieldtype: 'cone'箭头场，6×6 网格
particlestype: 'scatter3d'粒子散点，随机初始化 + 位移

交互功能：鼠标旋转、缩放、平移（Plotly 内置），参数滑块实时更新图形。

3. AI 讲解面板

  • 公式以 KaTeX 渲染（支持完整 LaTeX）
  • AI 返回中文讲解，含公式含义、物理意义、可视化要点
  • 支持 explain 接口独立调用（带上下文）

4. 预置公式库

8 个预设公式，点击即触发可视化：

公式类型
z = sin(x) * cos(y)3D 曲面
z = x² + y²3D 旋转抛物面
x=sin(3t), y=cos(5t)Lissajous 曲线
F = -kx (弹簧)弹性力学
E = mc²质能等价
F = (-y, x) 向量场旋转向量场
y = A sin(kx - wt)波动方程
抛物运动抛体运动

5. 设置与配置

设置面板（⚙ 按钮）可在线修改 AI 通道配置：

字段说明
Anthropic API Key支持 sk- 和 tp- 格式
Anthropic Base URL完整地址（如 https://xxx.com/anthropic/v1）
Anthropic Model模型名（如 mimo-v2.5）
OpenAI API Key支持 sk- 格式
OpenAI Base URL留空则用官方 API
OpenAI Model留空则默认 gpt-4o

保存后写入 backend/.env，立即生效，无需重启后端。

通道状态检测：侧边栏按钮显示可用状态，未配置的通道自动置灰禁用。

模型列表：/api/models 自动从 ANTHROPIC_BASE_URL 推导 /v1/models 端点，获取模型列表并过滤 TTS 等非文本模型。

6. 加载状态

AI 解析期间（约 5-15s），画布区域显示加载遮罩：

  • 旋转 spinner
  • 滑动进度条动画
  • 提示文字 “AI 正在解析公式并生成可视化…”
