2. Layer 1: Core Modules (MCM/ICM Competition)

2.1 config.py — Global Parameters

  • 10 real asteroids with JPL orbital data (Bennu, Ryugu, Psyche, Ceres, Pallas, Phaethon, Hektor, etc.)
  • Economic parameters: P0=$50K/kg, Q0=200t/yr, ε=0.5
  • Q-learning: α=0.1, γ=0.95, ε_greedy=0.1, 5000 episodes

2.2 module1_astrodynamics.py — 3D Astrodynamics

  • vis-viva equation: V = √(GM(2/r - 1/a))
  • 3D cosine rule for plane-change Δv: Δv = √(V₁² + V₂² - 2V₁V₂ cos(i))
  • Synodic period: S = 1 / |1/T₁ - 1/T₂|
  • Golden windows: sinusoidal launch opportunity detection
  • Output: Δv cost matrix (10 asteroids × 21 years)

2.3 module2_market_shock.py — Market Dynamics

  • Inverse demand function: P(Q) = P₀ × (Q₀/(Q₀+Q))^ε
  • Supply ramp-up: logistic curves per asteroid
  • Price elasticity sweep: ε = 0.3 to 0.8

2.4 module3_game_theory.py — Duopoly Game Theory

  • Cournot-Nash equilibrium: simultaneous production competition
  • Stackelberg equilibrium: sequential leader-follower game
  • Time-series analysis: equilibrium evolution over 21 years
  • SpaceX (leader) vs Blue Origin (follower) model

2.5 module4_qlearning.py — Reinforcement Learning (3 backends)

BackendImplementationSpeed
GPU (CuPy)Batched environment, 100 episodes parallel~431K eps/s
CPU (Numba JIT)@njit compiled inner loop~15K eps/s
numpyPure Python fallback~1K eps/s
  • State space: (year × price_level) = 21 × 5 = 105 states
  • Action space: 5 quantity levels [0%, 25%, 50%, 75%, 100%]
  • Bellman update: Q(s,a) ← Q(s,a) + α·[R + γ·max Q(s’,·) - Q(s,a)]
  • ESS (Evolutionary Stable Strategy) extraction

2.6 module4_qlearning_gpu.py — GPU Parameter Sweep

  • 128 agents simultaneously on RTX 3060 (6GB VRAM)
  • Hyperparameter grid: α×γ×ε_greedy×demand_ε = 4×4×4×2 = 128 combinations
  • Expanded state: 50 price levels × 21 years = 1,050 states
  • Expanded actions: 20 quantity levels
  • 99.9% GPU utilization achieved
  • Throughput: 431,000 eps/s (141× faster than CPU)

6. Key Mathematical Models

Game Theory

  • Cournot Nash: simultaneous quantity competition, best-response iteration
  • Stackelberg: sequential leader-follower, first-order conditions
  • Multi-player Nash: capacity-weighted Cournot with 7+ players

Reinforcement Learning

  • ε-greedy: explore (random) vs exploit (argmax Q)
  • Bellman update: Q(s,a) ← Q(s,a) + α·[R + γ·max Q(s’,·) - Q(s,a)]
  • Multi-agent: 128 independent agents with different hyperparameters

Economics

  • Inverse demand: P(Q) = P₀ × (Q₀/(Q₀+Q))^ε
  • Wright’s Law: cost decline = C₀ × N^(-0.15)
  • Tatonnement: iterative price adjustment until supply = demand

Orbital Mechanics

  • Vis-viva: V = √(GM(2/r - 1/a))
  • Lambert: universal variable method for transfer orbit
  • Hohmann: minimum-energy transfer, Δv₁ + Δv₂

Physics

  • Relativistic rocket: v = c·tanh(Isp·g₀·ln(m₀/mf)/c)
  • Schwarzschild: GR correction factor = 1 - 1.5·GM/(c²r)
  • Stefan-Boltzmann: P = εσAT⁴ for thermal radiation
