一、公司概述

1.1 创立背景与使命

Anthropic PBC 于2021年由七名前OpenAI员工创立，总部位于美国旧金山。创始团队包括Dario Amodei（CEO，前OpenAI研究副总裁）、Daniela Amodei（总裁）、Jared Kaplan（首席科学官）、Jack Clark（政策负责人）、Chris Olah（可解释性研究先驱）等核心成员。

Anthropic创立的直接原因是创始团队对OpenAI发展方向的分歧——他们认为OpenAI在商业化进程中逐渐淡化了对AI安全的承诺。Anthropic的核心使命是构建安全、可靠、可解释的AI系统。

公司以公共利益公司（PBC）形式运营，在追求商业目标的同时，将社会利益作为法定目标之一。截至2026年5月，Anthropic的估值约为9650亿美元。

二、核心技术与模型

2.1 Claude 系列演进

Claude 是Anthropic的旗舰大语言模型系列，以数学家Claude Shannon命名：

  • Claude / Claude Instant（2023.03）：初代模型，仅限受邀用户使用。经过额外的安全测试后才发布。
  • Claude 2（2023.07）：首次向公众开放，能力全面提升。
  • Claude 3 系列（2024.03）：三个级别——Opus（最强）、Sonnet（平衡）、Haiku（最快速），分别面向不同使用场景。
  • Claude 3.5 Sonnet（2024.06）：在性能和效率上显著提升。
  • Claude 4 Opus 和 Sonnet（2025.05）：第四代模型，伴随MCP连接器等新API能力。Anthropic官方博客将其描述为”世界最佳编码模型”，SWE-bench得分分别为72.5%（Opus）和72.7%（Sonnet）。
  • Claude Opus 4.6 和 Sonnet 4.6（2026.02）：4.6代更新，分别于2026年2月5日和17日发布。
  • Mythos Preview / Project Glasswing（2026.04）：2026年4月7日宣布，作为”Project Glasswing”的一部分发布，是Anthropic最新的前沿模型。值得注意的是，Mythos并非计划向公众开放的通用模型，而是以预览形式面向企业级客户，暗示Anthropic正在采取”受控的仅企业部署”策略来管理其最强大模型的发布。

发布节奏加速趋势：从Claude 4（2025年5月）到Opus 4.6（2026年2月）间隔9个月，从Opus 4.6到Claude Opus 4.8（2026年5月28日）仅间隔约3个月——发布节奏明显加速，表明Anthropic正在全力追赶竞争前沿。

2.2 Constitutional AI（宪法AI）

Constitutional AI（CAI） 是Anthropic最具标志性的技术创新，也是Claude模型的核心对齐方法。

与传统的RLHF（基于人类反馈的强化学习）不同，CAI的核心思想是为AI制定一套”宪法”——一组行为准则和原则，AI系统据此自我评估和调整输出。Claude 2的宪法包含了来自1948年《世界人权宣言》和Apple服务条款等文件的原则。例如，其中一条规则是：“请选择最支持和鼓励自由、平等和兄弟情谊的回应。”

CAI的流程：AI生成输出 → AI自身基于宪法评估输出 → 调整模型以更好地遵循宪法。这种方法减少了对人类标注的依赖，同时提高了AI行为的可预测性和可解释性。

2.3 可解释性（Interpretability）研究

Anthropic在AI可解释性研究领域处于全球领先地位：

  • 特征发现：2024年，使用”字典学习”（dictionary learning）技术，在Claude中识别出数百万个神经激活模式（对应具体概念），例如与”金门大桥”相关的特征。
  • 多语言处理机制：2025年3月研究发现，多语言大模型部分地在概念空间中处理信息，然后转换为具体语言。
  • 提前规划能力：研究发现Claude在写诗时会先识别潜在的押韵词，然后生成以此结尾的诗行，表明LLM具有某种提前规划的能力。

2.4 MCP（Model Context Protocol）

2024年发布的MCP是Anthropic推出的开放协议，旨在标准化AI模型与外部工具和数据源的交互方式。MCP允许开发者将各种数据源、工具和API以统一方式连接到Claude，扩展了AI的应用边界。

2.5 Claude Code 与 Cowork

  • Claude Code：命令行AI编程代理，2025年5月从研究预览转为正式发布。标志着”vibe coding”（用自然语言描述需求让AI写代码）的流行。
  • Cowork：Claude Code的图形界面版本，面向非技术用户，支持自动化文件和任务管理。

2.6 产品生态

  • Claude.ai：Web界面，提供免费和付费服务。
  • API：面向开发者的模型API。
  • Amazon Bedrock：通过AWS平台访问Claude模型。
  • iOS应用和 Mac/Windows桌面应用。

3.2 关键技术贡献

  • Constitutional AI：开创了基于原则的AI对齐范式，为整个行业提供了RLHF之外的替代方案。
  • 机械可解释性（Mechanistic Interpretability）：Anthropic在理解神经网络内部运作方面做出了系统性贡献，推动了”打开黑箱”的研究方向。
  • AI安全量化框架：提出了AI Safety Levels（ASL）框架，类似于生物安全等级，为不同能力水平的AI系统制定相应的安全要求。
  • Responsible Scaling Policy：负责任扩展政策，根据模型能力等级设定不同的安全和部署要求。

四、政策与治理

4.1 安全优先理念

Anthropic将AI安全作为公司运营的核心原则，而非附加考虑。这体现在：

  • Responsible Scaling Policy（负责任扩展政策）：根据模型能力设定分级安全要求。当模型达到新的能力阈值时，自动触发更严格的安全评估和部署限制。
  • AI Safety Levels（ASL）框架：类似于生物安全等级制度，从ASL-1到ASL-4，每个等级有明确的安全标准。
  • 使用政策：明确禁止将Claude用于国内监控和完全自主武器。

4.2 与美国国防部的冲突（2026年）

这是Anthropic历史上最具标志性的政策事件：

  • 2025.11：Anthropic与Palantir和AWS合作向美国情报和国防机构提供Claude。
  • 2026.02.24：美国国防部长Pete Hegseth要求Anthropic在3天内移除AI使用限制，否则将动用《国防生产法》强制执行。
  • 2026.02.26：Anthropic拒绝五角大楼的要求，坚持保留禁止国内监控和完全自主武器的条款。
  • 2026.02.27：特朗普总统命令联邦政府机构停止使用Anthropic的模型。
  • 2026.03.05：五角大楼正式将Anthropic列为”供应链风险”，禁止军事承包商与其合作。
  • 2026.03：Anthropic起诉国防部违宪报复。Google、Amazon、Apple、Microsoft领导层支持诉讼。
  • 2026.03.26：联邦法官发出临时禁令，裁定五角大楼的行为”显然是经典的第一修正案报复”。

这一事件成为AI公司与政府权力之间关系的标志性案例，被广泛视为Anthropic坚持安全原则的证明。

4.3 政治参与

  • 2026.02：向非营利组织Public First Action捐赠2000万美元，该组织宣布将支持支持AI监管的政治候选人。
  • Anthropic Institute（2026.03）：成立AI政策研究智库，由Jack Clark领导。

4.4 版权与法律

  • 音乐出版商诉讼（2023.10）：Concord、Universal等起诉Claude输出受版权保护的歌词。
  • 版权集体诉讼（2024.08）：作家起诉Anthropic使用盗版副本训练模型。2025年6月法院裁定合法购买书籍用于训练属”合理使用”，但使用盗版副本不构成合理使用。
  • 15亿美元和解（2025.09）：史上最大版权和解，每本书约3000美元加利息。
  • Reddit诉讼（2025.06）：指控Anthropic未经同意使用用户数据训练模型。
  • Project Panama（2026.01曝光）：Anthropic大规模购买二手书、拆解扫描后回收纸张的机密项目，被描述为”摧毁性地扫描世界上所有书籍”的努力。法院裁定销毁合法购买的书籍属合理使用。

六、人文与社会影响

6.1 “AI安全公司”的品牌定位

Anthropic在行业中独树一帜地以”AI安全公司”自居。这一定位在OpenAI安全人才流失后进一步强化——Jan Leike和John Schulman等前OpenAI安全团队核心成员加入Anthropic，被视为”安全派”向Anthropic的迁移。

6.2 伦理坚守

2026年拒绝五角大楼要求的事件，使Anthropic成为AI伦理领域的标志性公司。联邦法官裁定五角大楼的行为构成”第一修正案报复”，进一步巩固了Anthropic在公众心目中的正面形象。

6.3 自动化与就业

2025年9月，Anthropic发布的报告显示，企业主要将AI用于自动化而非协作，四分之三使用Claude的企业将其用于”完全任务委托”。CEO Dario Amodei在同年预测AI将”消灭白领工作，尤其是金融、法律和咨询领域的入门级职位”。

6.4 Vibe Coding 的推动

Claude Code的推出使”vibe coding”（用自然语言描述需求让AI写代码）成为流行趋势。这个术语由AI研究者Andrej Karpathy（Anthropic研究员）于2025年2月提出，并随着AI编程工具的普及而迅速流行。

6.5 超级碗与公众形象

2026年2月，Anthropic在超级碗LX投放了两则广告，作为”A Time and a Place”营销活动的一部分。每则广告描绘AI助手在对话中突然转向推销虚构产品的场景。Anthropic声明Claude将保持无广告，与OpenAI在ChatGPT免费版中引入广告形成鲜明对比。

七、未来展望

7.1 发展方向

  • IPO：2026年秋季可能的公开上市将进一步提升资本实力和品牌影响力。Series H的9650亿美元估值为IPO奠定了基础。
  • 安全与能力的平衡：Anthropic的长期战略核心是在保持安全领先的同时不断提升模型能力。发布节奏的加速（从9个月缩短到3个月）表明公司正在全力提升能力维度。
  • Mythos与受控部署：Project Glasswing暗示Anthropic正在采取”受控的仅企业部署”策略来管理其最强大模型，而非追求全面的公众覆盖。
  • 企业市场：Claude Enterprise和API业务的持续扩展。
  • AI代理：Claude Code和MCP生态代表了从对话AI向自主代理的转变。
  • IPL（Anthropic Institute）：政策研究和AI治理方面的影响力提升。

7.2 挑战

  • 政府关系：与美国国防部的冲突可能影响政府合同和政治关系。
  • 人才竞争：在快速扩张中保持创业文化和安全导向。
  • 版权风险：Project Panama等操作的法律风险持续存在。
  • 规模化挑战：从9650亿美元估值向更高目标迈进需要持续证明商业能力。

7.3 机遇

  • 安全品牌溢价：在AI信任度成为核心竞争力的时代，安全声誉是巨大资产。
  • 企业合规需求：越来越多的企业需要安全可靠的AI解决方案。
  • IPO红利：公开市场融资将提供更大的增长空间。
  • 多云策略：同时与AWS、Google Cloud、Azure合作的灵活策略降低了对单一供应商的依赖。
