AI Computer Homework Auto-Completion System

Technical plan for a system that parses programming assignments, generates code via LLM, compiles, runs, and verifies — end to end.

AI 计算机作业自动完成系统

端到端技术方案：解析编程作业 → LLM 生成代码 → 编译运行 → 自动验证正确性。

calendar_today 2026-05-05 schedule ~10 min read

Overview

Automatically analyze computer science homework (PDF/Word/screenshots/text), identify requirements and programming language (C/C++/Java/Python), generate code via LLM, compile and run, then verify correctness — all through a web interface.

项目概述

自动分析计算机课作业（PDF/Word/截图/文本），识别任务要求与编程语言，调用大模型生成代码，自动编译运行并验证正确性——全程通过网页界面操作。

Task Analyzer

LLM analyzes parsed text and outputs structured JSON:

{
  "language": "python",
  "requirements": "Implement a calculator...",
  "input_spec": "Two integers",
  "output_spec": "Sum, difference, product, quotient",
  "test_cases": [
    {"input": "3 5", "expected_output": "8 -2 15 0.6"}
  ],
  "constraints": ["No built-in math lib"],
  "deadline": "2026-05-10"
}

Code Generator

  • LLM API (OpenAI / Anthropic / compatible)
  • Step-by-step: skeleton → core logic → edge cases
  • Self-check: LLM reviews and revises its own output
  • Multi-file projects (e.g. Java multi-class)

fact_check

Verification Engine

For each test case:
  1. Compile code (if needed)
  2. Feed input
  3. Capture stdout + stderr
  4. Compare with expected_output
  5. Report: PASS / FAIL / RUNTIME_ERROR
  6. If FAIL → feedback → regenerate → retry (max N rounds)
